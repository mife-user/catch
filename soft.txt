请用 Go 语言开发一个本地文件夹内容搜索命令行工具，要求如下：
 
1. 功能
​
- 在目标文件夹下打开终端，执行命令即可使用。
​
- 支持按关键字递归/非递归搜索当前目录下的文件内容与文件名。
​
- 内部使用协程池（goroutine pool）控制并发，避免资源占用过高。
​
- 提供可视化交互式命令行界面，类似 Claude CLI 风格，用户可通过方向键/数字选择功能。
​
- 支持搜索结果高亮、分页展示。
​
2. 使用方式
​
- 编译后可在当前目录直接运行，无需复杂安装。
​
- 提供选项：一键将工具添加到系统环境变量，实现全局调用。
​
3. 发布与部署
​
- 项目结构清晰，可直接上传 GitHub 供他人下载使用。
​
- 提供简单的  README ，包含：下载、编译、运行、添加到环境变量、使用说明。
​
4. 技术要求
​
- 纯 Go 实现，不依赖多余第三方库（可使用轻量 CLI 交互库）。
​
- 协程池固定数量协程处理文件遍历与内容搜索。
​
- 跨平台兼容（Windows / macOS / Linux）。
